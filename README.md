# CSS Architecture

> OOCSS, SMACSS and BEM

## How to setup

- vscode plugin installation

  - `live sass compiler` - `Ritwick`

- live sass compiler configuration

  ```json
  "liveSassCompile.settings.formats": [
    {
      "format": "expanded",
      "extensionName": ".css",
      "savePath": "/dist/css"
    }
  ]
  ```

- start the plugin
